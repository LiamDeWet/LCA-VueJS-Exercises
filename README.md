## Overview

This project is a Vue 3 single-page application that displays cooking courses offered by Cooking Masterclass.

Users can:

- View available courses
- See chef names
- View course levels
- View pricing
- Identify sold out classes
- Save courses to a wishlist

## Technologies Used

- Vue 3
- Vite
- CSS

## Installation
### Clone the repository to you machine then run these commands: 
```bash
npm install
npm run dev
```
